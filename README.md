# toast-queue

> A Vanilla JS library to display brief, temporary notifications.

toast-queue is a headless, accessible notification library to display brief, temporary toast messages for the web.

- Headless architecture: Complete styling control.
- [View Transition API](https://developer.mozilla.org/en-US/docs/Web/API/View_Transition_API) ready: Smooth animations with modern browser features.
- Touch-friendly swiping: Native gesture support for dismissing notifications.
- Accessibility built-in
- Framework agnostic (Vanilla JS, no dependencies)
