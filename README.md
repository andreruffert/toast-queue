# toast-queue

> A unstyled, accessible Vanilla JS library to display brief, temporary toast messages.

- Framework agnostic: Vanilla JavaScript without dependencies.
- Headless UI: Complete styling control without any pre-defined visual styles.
- Accessibility built-in
- [View Transition API](https://developer.mozilla.org/en-US/docs/Web/API/View_Transition_API) ready: Smooth animations with modern browser features.
- Touch-friendly swiping: Native gesture support for dismissing notifications.
